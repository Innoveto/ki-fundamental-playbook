## Kernaussage

Mit den Texten aus Rezept 3 und den Bildern aus Rezept 4 baust du in Replit eine komplette, gehostete Landing Page. Kein Code nötig, aber du siehst den Code entstehen.

## Lernziel

Du hast am Ende dieses Blocks eine live erreichbare Landing Page für dein Startup.

## Ablauf

### Demo

Workflow-Überblick: Text (Claude) + Struktur (Claude) + Visuals (Bildgenerierung) + Build (Replit) = Live-Seite

### Hands-on

> [!NOTE]
> Du brauchst die Texte aus Rezept 3 und die Bilder aus Rezept 4. Halte die Ergebnisse aus den vorherigen Blöcken bereit.

1. Öffne Replit und starte den AI Agent
2. Beschreibe deine Seite in 2-3 Sätzen (Texte aus Rezept 3 einfliessen lassen)
3. Lass den Agent die Seite generieren
4. Pass Design und Inhalte an
5. Publiziere auf replit.dev
6. Wenn du früh fertig bist: Bilder aus Rezept 4 einbauen, weitere Texte verfeinern
