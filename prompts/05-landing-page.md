> Alle Prompts zum Kopieren und Anpassen. Du brauchst dafür die Texte aus Rezept 3 und die Bilder aus Rezept 4.

## Schritt 1: Replit-Agent-Prompt (Komplett)

**Wofür:** Eine komplette Landing Page mit dem Replit AI Agent generieren
**Tool:** Replit

```
Erstelle eine moderne Landing Page für mein Startup "[STARTUP-NAME]".

Das Produkt: [1-2 SÄTZE ZUM PRODUKT]
Zielgruppe: [ZIELGRUPPE]

Die Seite soll folgende Sektionen haben:
1. Hero mit Headline, Subheadline und CTA-Button
2. Features (3 Hauptvorteile)
3. So funktioniert's (3 Schritte)
4. Testimonials / Social Proof
5. FAQ (4 Fragen)
6. Footer mit CTA

Design: Modern, clean, mit den Farben [FARBE 1] und [FARBE 2].
Schriftart: Sans-serif, gut lesbar.
Mobile-optimiert.

Texte:
Hero-Headline: [DEINE HEADLINE AUS REZEPT 3]
Hero-Subheadline: [DEINE SUBHEADLINE AUS REZEPT 3]
CTA-Text: [DEIN CTA-TEXT]
```

**Was du bekommst:** Eine vollständige, funktionierende Landing Page mit allen Sektionen. Der Replit Agent baut die Seite automatisch auf.

> [!NOTE]
> Setze deine fertigen Texte aus Rezept 3 direkt ein. So sparst du dir Anpassungen danach.

## Schritt 2: Replit-Agent-Prompt (Minimal)

**Wofür:** Schnellster Weg zu einer funktionierenden Seite
**Tool:** Replit

```
Erstelle eine simple, moderne Landing Page für "[STARTUP-NAME]": [PRODUKTBESCHREIBUNG IN EINEM SATZ]. Farben: [FARBE 1] und [FARBE 2]. Hero, Features, CTA, Footer. Mobile-optimiert.
```

**Was du bekommst:** Eine einfachere Seite mit weniger Sektionen. Gut, wenn du schnell ein Ergebnis willst und danach anpassen möchtest.

## Schritt 3: Design-Anpassungen

**Wofür:** Nachdem die Seite generiert ist, Design verfeinern
**Tool:** Replit

```
Ändere folgendes an der Landing Page:
- [ÄNDERUNG 1, z.B. "Mach die Hero-Section grösser"]
- [ÄNDERUNG 2, z.B. "Ändere die Hintergrundfarbe zu #F5F5F5"]
- [ÄNDERUNG 3, z.B. "Füge ein Bild in der Hero-Section hinzu"]
```

**Was du bekommst:** Die aktualisierte Seite mit deinen Änderungen. Du kannst diesen Schritt beliebig oft wiederholen.

> [!TIP]
> Ändere immer nur 2-3 Dinge auf einmal. Kleine Schritte geben dir mehr Kontrolle über das Ergebnis.
