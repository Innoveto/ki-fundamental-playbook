# Registrierung

[← Zurück zum Playbook](../README.md)

---

Registriere dich bei den folgenden Tools. Alle sind kostenlos, keine Kreditkarte nötig.

## Kerntools

- [ ] **Claude Free** auf [claude.ai](https://claude.ai) registrieren (Email oder Google Login)
- [ ] **Canva Free** auf [canva.com](https://canva.com) registrieren
- [ ] **Replit** auf [replit.com](https://replit.com) registrieren
- [ ] **Kling AI** auf [klingai.com](https://klingai.com) registrieren
- [ ] **Hailuo AI** auf [hailuoai.video](https://hailuoai.video) registrieren
- [ ] **Haiper** auf [haiper.ai](https://haiper.ai) registrieren

## Empfohlen

- [ ] **Ideogram** auf [ideogram.ai](https://ideogram.ai) registrieren
- [ ] **Microsoft Account** für [Copilot](https://copilot.microsoft.com) (falls noch keiner vorhanden)
- [ ] **Gamma.app** auf [gamma.app](https://gamma.app) registrieren

## Hinweise

- Browser: Chrome oder Edge empfohlen (manche Tools funktionieren in Safari eingeschränkt)
- Die Video-Tools (Kling, Hailuo, Haiper) haben tägliche oder monatliche Limits. Details: [tools.md](tools.md)
