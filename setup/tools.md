## Kerntools

Alle kostenlos, keine Kreditkarte nötig.

### 1. Claude Free

| | |
|---|---|
| **URL** | [claude.ai](https://claude.ai) |
| **Wofür** | Chat, Prompting, Texte, Analyse, Daten-Upload |
| **Free Tier** | Ca. 40 Nachrichten/Tag (Rolling Reset) |
| **Anmeldung** | Email, Google oder Microsoft |
| **Features** | Web-Suche, Datei-Upload, Artifacts |

### 2. Canva Free

| | |
|---|---|
| **URL** | [canva.com](https://canva.com) |
| **Wofür** | Social Media Design, Präsentationen, Brand Kit |
| **Free Tier** | Unbegrenzt (1.8M+ Templates) |
| **Anmeldung** | Email oder Google |
| **Features** | Templates für alle Kanäle, 3 Brand-Farben, 5 GB Storage |

### 3. Video: 3 Optionen

Für Video hast du 3 Tools zur Auswahl. Probier am besten 2 davon aus und vergleich die Ergebnisse.

| | Kling AI | Hailuo AI | Haiper |
|--|----------|-----------|--------|
| **URL** | [klingai.com](https://klingai.com) | [hailuoai.video](https://hailuoai.video) | [haiper.ai](https://haiper.ai) |
| **Free Credits** | 66/Tag (Reset täglich) | ~20-30 Clips/Monat | 300 einmalig |
| **Videos/Session** | ~3-4 Clips | ~3-5 Clips | ~6 Clips |
| **Auflösung** | 720p | 1080p | 720p |
| **Stärke** | Meiste Features | Beste Qualität | Einfachste Bedienung |

### 4. Replit

| | |
|---|---|
| **URL** | [replit.com](https://replit.com) |
| **Wofür** | Landing Page aus Textbeschreibung generieren |
| **Free Tier** | 10 Apps, 1.200 Dev-Minuten/Monat, Hosting inklusive |
| **Anmeldung** | Email, Google oder GitHub |
| **Features** | AI Agent generiert aus Prompt, Code sichtbar |
